OpenWrt / ImmortalWrt ImageBuilder

使用说明：

  1. 选择分支 (OpenWrt 或 ImmortalWrt)
  2. 输入设备名称 (如: x86-64, rockchip-armv8)
  3. 选择版本号
  4. 配置其他参数（可选）
  5. 点击 "Create Issue" 提交构建请求

常用设备名称：

  • x86-64: 通用 x86 64位设备
  • rockchip-armv8: Rockchip ARM 设备
  • ramips-mt7621: MediaTek MT7621 设备
  • ath79-generic: Atheros AR71xx/AR9xxx 设备
