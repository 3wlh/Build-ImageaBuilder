ImageBuilder

使用说明：

  1. 选择分支 (OpenWrt 或 ImmortalWrt)
  2. 选择设备架构（从 JS 文件读取）
  3. 选择具体型号（根据架构自动更新）
  4. 选择版本号（根据分支自动更新）
  5. 点击选择需要的插件包和主题
  6. 输入自定义脚本（可选）
  7. 配置其他参数（可选）
  8. 点击 "Create Issue" 提交构建请求

数据来源：

  • 设备列表：js/device.js (image 对象)
  • 版本列表：js/version.js (versions 对象)
  • 插件包列表：js/packages.js

设备格式：

  • image 参数：架构名称（如 x86_64, rockchip-armv8）
  • device 参数：设备型号（如 generic, friendlyarm_nanopi-r6s）
  • 示例：image="x86_64", device="generic"
